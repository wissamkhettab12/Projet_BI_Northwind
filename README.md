# Projet_BI_Northwind
Projet BI sur la base Northwind avec Power BI 
## 1. Description du projet
Ce projet académique consiste à concevoir une solution complète de Business Intelligence basée sur la base de données Northwind.
Il met en œuvre un processus ETL réalisé avec Power BI, une modélisation dimensionnelle en étoile et des visualisations analytiques avancées avec Python.

## 2. Arborescence du projet
Projet_BI_Northwind/
- data/
- scripts/
- reports/
- figures/
- video/
- powerbi/

## 3. Contenu des dossiers
data/ : bases de données Northwind (SQL Server) et fichier Access complémentaire  
scripts/ : scripts Python pour visualisation et analyse  
reports/ : rapport académique final (PDF)  
figures/ : graphiques générés avec Python  
video/ : vidéo de présentation du projet  
powerbi/ : fichier Power BI (.pbix)

## 4. Pré-requis
- Power BI Desktop
- SQL Server (base Northwind)
- Microsoft Access
- Python 3 (optionnel)
- Bibliothèques Python : pandas, matplotlib, seaborn, plotly

## 5. Exécution des scripts Python
1. Ouvrir le dossier du projet
2. Installer les bibliothèques :
pip install pandas matplotlib seaborn plotly
3. Exécuter les scripts sur power bi
Les graphiques sont générés dans le dossier figures/.

## 6. Power BI
Ouvrir le fichier .pbix dans le dossier powerbi/ avec Power BI Desktop, puis actualiser les données.

## 7. Justification des choix techniques
Power BI a été utilisé pour l’ETL, la modélisation et la création des tableaux de bord.
Python a été choisi pour ses capacités d’analyse et de visualisation.
Pandas permet la manipulation efficace des données.
Matplotlib, Seaborn et Plotly sont utilisés pour créer des graphiques clairs et interactifs.
